# Conversations Directory

This directory stores timestamped conversation logs.

## Filename Format

`YYYY-MM-DD_HH-MM-SS.md`

Example:
`2026-03-17_21-35-02.md`

## Suggested Log Format

- session metadata
- selected role
- user messages
- assistant responses
- optional future structured observations:
  - pain score
  - gesture input
  - image/VLM observations
